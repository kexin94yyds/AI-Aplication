# 快速启动指南

## 新功能速览 🎉

本次更新添加了两个重要功能：

1. **🖱️ 窗口自由拖动** - 不再锁定在屏幕右侧，可以随意移动
2. **🔝 窗口真正置顶** - 点击右上角按钮，让侧边栏**真正**悬浮在所有应用之上（包括全屏应用！）

## ⚡ v2.0 重大升级

参考了 [RI 项目](https://github.com/kexin94yyds/RI.git) 的实现，置顶功能现在使用 `screen-saver` 层级，可以：

- ✅ 悬浮在全屏浏览器之上
- ✅ 悬浮在全屏视频之上
- ✅ 悬浮在演示文稿之上
- ✅ 真正的"始终在最前面"

## 启动应用

### 方式一：普通启动
```bash
cd "/Users/apple/全局 ai 侧边栏/AI-Sidebar"
npm start
```

### 方式二：开发模式（带开发者工具）
```bash
npm run dev
```

## 快速测试

启动后：

1. **测试拖动功能**
   - 在窗口顶部按住鼠标左键
   - 拖动到任意位置
   - 松开鼠标

2. **测试置顶功能**
   - 点击右上角的 🔝 按钮
   - 打开其他应用（如浏览器）
   - 观察侧边栏是否保持在最上层
   - 再次点击 🔝 取消置顶

## 快捷键

- `Alt+Space` (或 `Option+Space`) - 显示/隐藏侧边栏
- 其他快捷键在 Settings 中可以自定义

## 视觉标识

- **紫色渐变** 🔝 = 未置顶
- **粉色渐变** 🔝 = 已置顶

## 常见问题

**Q: 为什么看不到置顶按钮？**
A: 确保使用的是 Electron 应用，不是浏览器扩展。

**Q: 拖动没有反应？**
A: 确保在窗口顶部区域（前40px高度）拖动，避开置顶按钮。

**Q: 置顶后其他窗口还是会遮挡？**
A: 检查按钮是否变成粉色渐变，如果没有变化，请重新点击。
